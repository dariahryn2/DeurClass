# DeurClass
Class Door vertegenwoordigt een deur met een naam, vergrendelingsstatus en open/gesloten status.  
  
UseKey(): "links" ontgrendelt de deur, "rechts" vergrendelt de deur  
OpenDoor(): opent de deur als deze ontgrendeld is  
WalkThrough(): laat je erdoorheen lopen als de deur open is, anders bots je ertegenaan
